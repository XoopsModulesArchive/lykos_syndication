Lykos Syndication Module

Written by Samuel Wright (http://www.lykoszine.co.uk) 
Some code adapted from backend.php (http://www.xoops.org)

This module completely replace my javascript-backend hack.

----==== Requirement ====----

Xoops RC3.05 or Greater

----==== Installing ====----

1. Upload the folder inside the zip to the Xoops modules folder.
2. Login to your site as Admin, In the Admin area of xoops goto 'modules' and use the pull down menu dialog and select install.
Then follow the instruction until finished.
